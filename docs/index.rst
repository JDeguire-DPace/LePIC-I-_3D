LePIC 3D documentation
======================

LePIC 3D is a modular Fortran Particle-in-Cell code for plasma simulations.
This documentation describes how the repository is organized, how to build and run the code, and where the main modules live.

.. note::

   This is a starter documentation tree. It is intentionally simple so it can build cleanly on Read the Docs first.
   After that, API-level Fortran documentation can be added progressively.
