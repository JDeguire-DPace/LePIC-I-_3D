LePIC Documentation
===================

LePIC is a modular Particle-In-Cell plasma simulation code written in modern Fortran.

Features
--------

* Electrostatic PIC solver
* Modular OOP architecture
* MPI/OpenMP support
* Plasma chemistry and collisions
* RF ion source simulations
